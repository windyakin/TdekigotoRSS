# TdekigotoRSS
津山高専の[最近のできごと](http://www.tsuyama-ct.ac.jp/dekigoto.htm)をクロールしてRSS形式で出力します。

## Usage
cpanmを使って必要なモジュールを``perllib``以下にインストールして下さい(環境によっては不要)。

```sh
% cpanm -l perllib local::lib
% cpanm -l perllib Template::Extract
% cpanm -l perllib XML::FeedPP
```
